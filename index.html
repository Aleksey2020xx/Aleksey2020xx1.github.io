О себе

Опыт работы более года. Начинал осваивать верстку и JavaScript самостоятельно, позже приобрел дополнительно курсы от школы Skillbox по направлению web-разработчик, для закрытия "пробелов" в знаниях. За это время собрал портфолио (ссылка на github выше), изучил HTML (PUG), CSS (SCSS/bootsrap), JavaScript, Git, Gulp. Постоянно занимаюсь практикой, закрепляю полученные знания, осваиваю новые навыки, развиваюсь в сторону фронтенд-разработки. Имею опыт коммерческой разработки - являюсь участником стартапа, на позиции верстальщик, который запускает сервис по разработке и продвижению сайтов, но в связи с долгим и непродуктивным развитием компании, принял решение покинуть команду. В данный момент ищу новое место работы, где буду иметь возможность роста как фронтенд-разработчик, принимать участие в разработке сложных и интересных проектов в команде профессионалов.
